PROMPTIES
=========

Here I collect Fun, Helpful, Creative Prompts I find all over the internets. 

## How to use

I group things together. In each folder you'll find markdown files with more details and the prompt.
**Always** consider adjusting the prompt to your liking needs. Stuff that should be adjusted for sure are marked with `{hint}`.
