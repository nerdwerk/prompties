# Cartoon Style Avatar

Turn a photo into a specific cartoon style.

## The Prompt

_Use with an Image-to-Image model._

```markdown
Transform this reference photo into a character in the style of {hint: e.g., Pixar, Studio Ghibli, 90s Anime, The Simpsons}.

**Details:**

-   **Subject**: Maintain the subject's gender, age, and key distinctive features (glasses, beard, hair color).
-   **Style Elements**:
    -   _If Pixar_: 3D render, soft lighting, big eyes, vibrant colors.
    -   _If Ghibli_: Hand-drawn 2D, watercolor backgrounds, soft aesthetic.
    -   _If Anime_: Sharp lines, cel-shaded, expressive eyes.
-   **Background**: {hint: Describe desired background or keep simple}.
```
