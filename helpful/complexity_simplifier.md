# Complexity Simplifier (ELI5)

Explain complex topics in simple terms.

## The Prompt

```markdown
I need to understand a complex topic. Please explain it to me as if I were {hint: a 5-year-old, a high school student, a non-technical manager}.

**Topic**: {hint: Quantum Computing, The Blockchain, How a Sewing Machine Works}

**Constraints:**

-   Use simple analogies.
-   Avoid jargon (or explain it if necessary).
-   Keep it under {hint: 200 words}.
-   Focus on the "Why" and "How" in basic terms.
```
